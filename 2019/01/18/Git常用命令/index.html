Git常用命令

  1. 初始化
    git init
  2. 添加
    git add
  3. 添加所有改动
    git add -A . or git add --all
  4. 删除
    git rm
  5. 查看状态
    git status
  6. 提交信息
    git commit -m "something"
  7. 提交日志
    git log
  8. 复制远程仓库
    git clone https://github.com/name/name.git
  9. 推送远程仓库
    git push origin master
  10. 推送并创建远程仓库
    git push -u origin master
  11. 关联远程仓库
    git remote add origin https://github.com/name/name.git
