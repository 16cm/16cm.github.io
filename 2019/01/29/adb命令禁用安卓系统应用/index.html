adb命令禁用安卓系统应用

无bl解锁/无root，禁用系统自带的无用软件
adb工具下载地址：Win Linux macOS

  1. 启用列表
    adb shell pm list packages -s -e >enabled.txt
    在adb目录下多了一个enabled.txt文件
  2. 禁用列表
    adb shell pm list packages -s -d >disabled.txt
    在adb目录下多了一个disabled.txt文件
  3. 禁用命令
    adb shell pm disable-user com.android.stk
  4. 批处理
    也可以使用批处理一次禁用多个软件，新建文本文档，添加以下内容
    adb shell pm disable-user com.android.stk1
    adb shell pm disable-user com.android.stk2
    adb shell pm disable-user com.android.stk3
    命令类似上面，将想禁用的软件添加进去，文件后缀改为bat，放到adb目录后双击运行。
