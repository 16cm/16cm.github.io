崂山一日游

2019-05-03

崂山游玩有北线、中线、南线，中线为巨峰，主要是爬山，北线为北九水，雨后去会比较好玩，南线从流清 - 太清 - 仰口 - 棋盘石，所以选了南线。

非自驾的话，建议早一些去坐公交，火车站附近可以在西陵峡路火车站坐304路公交到大河东客服中心，也可以坐地铁到石老人浴场转304路，下车进售票厅后右转排队买票，南线联票130元，包含90元门票+40元大巴，假期人多可能需要排队很久。

几种不同的标志符号CMC、CMA、CPA、CPE

2019-03-11

在法制计量管理工作中，经常会遇到CMC、CMA、CPA、CPE等标志符号，这些形相似、意不同、使用场合不一样的标志符号很容易使人混淆，需仔细辨认，区别使用。

下面分别阐述：

各标志符号的含义
  1. CMA - 计量认证合格 (Metrology Approval Certification) 标志

该标志仅适用于产品质量检验机构。凡要开展为社会提供公证数据的产品质量检验机构，均需向有关人民政府计量行政部门提出申请并进行考核，经考核合格的机构，由有关人民政府计量行政部门审查批准，颁发计量认证合格证书，并同意其使用统一的计量认证标志CMA，同时公布其机构的名称和检验范围。

Git常用命令

2019-01-18
  1. 初始化
    git init
  2. 添加
    git add
  3. 添加所有改动
    git add -A . or git add --all
  4. 删除
    git rm
  5. 查看状态
    git status
  6. 提交信息
    git commit -m "something"
  7. 提交日志
    git log
  8. 复制远程仓库
    git clone https://github.com/name/name.git
  9. 推送远程仓库
    git push origin master
  10. 推送并创建远程仓库
    git push -u origin master
  11. 关联远程仓库
    git remote add origin https://github.com/name/name.git

解决浏览器出现您是不是要访问的问题

2019-01-14

这个问题一般是 DNS 劫持，以下是解决方法：

  1. 在网络连接里面更换 DNS，可以设置为 114.114.114.114/114.114.115.115;
  2. 命令行窗口输入 ipconfig/flushdns 刷新;
  3. 在路由器 WAN 端修改 DNS。
