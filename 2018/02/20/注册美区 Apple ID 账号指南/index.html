注册美区 Apple ID 账号指南

苹果官网称，从2018年2月28日起，中国区iCloud账户将转由“云上贵州大数据产业发展有限公司”运营。
鉴于此，注册个Apple ID转战美区，以下方法不需绑定信用卡。

1、注册帐号

首先挂美国的代理，打开https://www.apple.com/，点击右边的购物袋，选择 Sign in ，点击右侧的 Don’t have an Apple ID? Create one now. ，然后依次填写信息，年龄填写20+，以免日后在 App Store 里面有些应用无法购买。
图中的地方为 Apple ID ，最好是谷歌或微软邮箱，并且是未绑定过 Apple ID 的邮箱。填写完点击最下方的 Continue ，将邮箱收到的验证码填写，确认后即注册成功。

2、设备登录帐号

iPhone上登录帐号之前挂上代理并且开启全局代理，然后在 App Store 里面注销原有账号后登录刚才注册的 Apple ID，这时会要求填写账单信息：
1.付款信息里面选择 None；
2.名字随便填写；
3.可以使用以下地址：
Infinite Loop
Cupertino, CA 95014
(408) 996–1010
填写完点击 Done 确认，这时可以去搜索任意一个软件下载试试。

3、购买应用

美区无法使用支付宝或微信支付，所以我们选择在苹果官网购买礼品卡的方法。
苹果有一个通过邮件方式购买App Store iTunes Gift Cards的服务，用户可以在苹果官网购买Gift Cards，通过邮件接收，无需实体卡邮寄，这样就可以实现美国区App Store的充值，而不必再绑定支付方式。
