基于Hexo插入图片的Markdown语法

Mar 24 2017

虽然在官方语法中，Markdown插入图片的格式是这样的：

![Alt text](img.jpg)

然而用了Hexo框架，则需要在 source 文件夹中新建一个 images 文件夹，然后将图片放入其中。

路径就变成这样：

![Alt text](/images/img.jpg)

不过有个缺点就是Markdown编辑器无法实时预览。
