Markdown语法

发表于 2017-01-10 |

一些常用的 Markdown 语法。

段落

    一行就是一个段落，如果要另起一行，在两个段落之间添加一个空行即可。

斜体

    *斜体*

    效果

    斜体

粗体

    **粗体**

    效果

    粗体

    ***斜体+粗体***

    效果

    斜体+粗体

上标

    n<sup>2</sup>

    效果

    n2

下标

    a=log<sub>2</sub>b

    效果

    a=log2b

函数

    &fnof;(x)=x+1

    效果

    ƒ(x)=x+1

根号

    &radic;5

    效果

    √5

角度

    30&deg;

    效果

    30°

标题

    # 这是 H1 #

    ## 这是 H2 ##

    ### 这是 H3 ###

    ...

    ###### 这是 H6 ######

    效果

这是 H1

这是 H2

这是 H3

…

这是 H6

引用

    >markdown 的语法简洁，但有其局限性，所以特意保留了内联html这种方式。任何html标签及其内容，都会原样输出到结果中。也就是说，标签中的星号等作为markdown结构的符号，以及构成html标签和实体的符号，都不会做任何转义。

列表

无序列表

    * 第一条
    * 第二条
    * 第三条

    效果

  • 第三条

    1. 第一条
    2. 第二条
    3. 第三条

效果

  2. 第二条
  3. 第三条

代码

    用反引号 ` 来标记内联代码，如果代码的内容中有反引号，请用两个反引号包裹。

    代码中的 &、<、> 符号都会自动转义，请放心使用。
