Markdown语法

一些常用的 Markdown 语法。

段落

一行就是一个段落，如果要另起一行，在两个段落之间添加一个空行即可。

斜体

*斜体* -> 斜体

粗体

**粗体** -> 粗体

***斜体+粗体*** -> 斜体+粗体

上标

n<sup>2</sup> -> n2

下标

a=log<sub>2</sub>b -> a=log2b

函数

&fnof;(x)=x+1 -> ƒ(x)=x+1

根号

&radic;5 -> √5

角度

30&deg; -> 30°

标题

# 这是 H1 #

## 这是 H2 ##

### 这是 H3 ###

...

###### 这是 H5 ######

这是 H1

这是 H2

这是 H3

…

这是 H5

列表

无序列表

* 第一条
* 第二条
* 第三条

  • 第一条
  • 第二条
  • 第三条
有序列表

1. 第一条
2. 第二条
3. 第三条

  2. 第二条

代码

用反引号 ` 来标记内联代码，如果代码的内容中有反引号，请用两个反引号包裹。

代码中的 &、<、> 符号都会自动转义，请放心使用。

代码区域

\代码区域
\

代码区域

http://123.com

http://123.com



物品 数量 单价
iPhone 1 100
iPad 2 200
IPod 3 300

```

效果

物品 数量 单价
iPhone 1 100
iPad 2 200
IPod 3 300

注意第二行的冒号表示居左居右还是居中。
