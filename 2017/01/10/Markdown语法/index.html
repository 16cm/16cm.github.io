Markdown语法

发表于 2017-01-10

一些常用的 Markdown 语法。

01、段落

一行就是一个段落，如果要另起一行，在两个段落之间添加一个空行即可。

02、斜体

*斜体* -> 斜体

03、粗体

**粗体** -> 粗体
***斜体+粗体*** -> 斜体+粗体

04、上标

n<sup>2</sup> -> n2

05、下标

a=log<sub>2</sub>b -> a=log2b

06、函数

&fnof;(x)=x+1 -> ƒ(x)=x+1

07、根号

&radic;5 -> √5

08、角度

30&deg; -> 30°

09、标题

# 这是 H1 #
## 这是 H2 ##
### 这是 H3 ###
...
###### 这是 H5 ######

11、列表

11.1无序列表

* 第三条

  • 第一条
  • 第二条
  • 第三条
11.2有序列表

1. 第一条
2. 第二条
3. 第三条

  1. 第一条
  2. 第二条
  3. 第三条

12、代码

用反引号 ` 来标记内联代码，如果代码的内容中有反引号，请用两个反引号包裹。
代码中的 &、<、> 符号都会自动转义，请放心使用。
