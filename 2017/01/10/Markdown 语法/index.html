Markdown 语法

发表于 2017-01-10 | | 阅读次数

一些常用的 Markdown 语法。

段落

 一行就是一个段落，如果要另起一行，在两个段落之间添加一个空行即可。

斜体

*斜体*

效果

粗体

**粗体**

效果

粗体

***斜体+粗体***

效果

斜体+粗体

上标

n<sup>2</sup>

效果

n2

下标

a=log<sub>2</sub>b

效果

a=log2b

函数

&fnof;(x)=x+1

效果

ƒ(x)=x+1

根号

&radic;5

效果

√5

效果

30°

代码

用反引号 ` 来标记内联代码，如果代码的内容中有反引号，请用两个反引号包裹。

代码中的 &、<、> 符号都会自动转义，请放心使用。
